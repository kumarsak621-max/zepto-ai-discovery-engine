# Generated / cached review data (runtime)

This folder holds:

- `reviews.csv` — Google Play cache (auto-created)
- `merged_reviews.csv` — merged multi-source dataset (auto-created)
- `live_reviews_meta.json` — last refresh metadata (auto-created)
- `zepto_reviews.csv` — optional sample dataset (committed)

These files are created automatically when you run **Review Analysis** or
**Refresh Live Reviews**. Do not upload CSVs manually.
