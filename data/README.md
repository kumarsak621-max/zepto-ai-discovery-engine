# Generated / cached review data (runtime)

This folder holds:

- `reviews.csv` — Google Play cache (auto-created)
- `merged_reviews.csv` — merged multi-source dataset (auto-created)
- `live_reviews_meta.json` — last refresh metadata (auto-created)
- `manual_reviews.csv` — normalized manual upload (auto-created after upload)
- `manual_reviews_meta.json` — manual upload metadata
- `discovery_insights_cache.json` — cached Gemini discovery dashboard payload
- `zepto_reviews.csv` — optional sample dataset (committed)

These files are created when you run **Review Analysis**, **Refresh Live Reviews**,
or upload a file via **📂 Upload Manual Reviews**.
